# learning-ai
A collection of **AI-based** works organized by sub-categories.

## Repository Format 
### Subcategories
The **subcategories** in this repository include: 
- Pre-Requisite Mathematics 
- Convolutional Networks 
- Deep Neural Networks 
- Fitting Models 
- Gradients and Initialization 
- Loss Functions 
- Measuring Performance 
- Regularization 
- Shallow Neural Networks 
- Supervised Learning 
- Transformers  

### Subfolders 
#### Summaries
Each subcategory contains a summary of topic, written in a markdown file. 

#### Code 
Each subcategory contains a subfolder called 'Code'. The subfolder contains Python code demonstrating a function prevalent in the subcategory. 

#### Jupyter Notebooks
Each subcategory also contains a subfolder called 'Jupyter Notebooks'. The subfolder contains Jupyter notebooks, which provides a detailed explanation of the code.
